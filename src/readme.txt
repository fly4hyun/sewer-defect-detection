#############################
# 서울디지털재단 하수관로 결함 탐지 프로젝트
#############################

## 파일 설명
### `sewer_simulation_api.py`
- 하수관로 이미지를 입력받아 결함 탐지 결과를 반환하는 API 코드.

### `track_api_new.py`
- 하수관로 영상을 입력받아 결함 탐지 결과를 반환하는 API 코드.

### `train.py`
- 하수관로 데이터셋을 이용해 YOLO 모델을 학습하는 코드.

### `test.py`
- 하수관로 데이터셋으로 학습된 YOLO 모델을 평가하는 코드.

### `track.py`
- 하수관로 영상에서 결함을 탐지하는 코드.

### `yolo_data_generate.py`
- 서울디지털재단 하수관로 데이터를 yolo 형식으로 변환하는 코드.

### `sewer_api.tar`
- api 사용을 위한 도커 환경 이미지 TAR 파일.

### `simulation_api_run.sh`
- 하수관로 이미지를 입력받아 결함 탐지 결과를 반환하는 API 실행 BASH 파일.

### `track_api_run.sh`
- 하수관로 영상을 입력받아 결함 탐지 결과를 반환하는 API 실행 BASH 파일.

## 폴더 구조
### `classifier_weights`
- 후처리 분류 모델의 가중치가 저장되는 폴더.

### `datasets`
- 학습 및 평가에 사용되는 데이터셋을 저장하는 폴더.

### `data`
- 하수관로 원천 데이터 및 데이터를 학습, 검증, 평가로 나눈 txt 파일을 저장하는 폴더 (서울디지털재단 데이터를 여기에 저장).

### `results`
- 모델 평가 결과가 저장되는 폴더.

### `utils`
- 유틸리티 함수를 포함하는 폴더.

### `video`
- 분석 대상 하수관로 영상을 저장하는 폴더.

### `video_test_image_results`
- 분석된 하수관로 영상의 결과를 이미지 단위로 저장하는 폴더.

### `video_test_results`
- 분석된 하수관로 영상의 결과를 저장하는 폴더.

### `weight`
- 학습된 하수관로 YOLO 모델의 가중치를 저장하는 폴더.

### `results_video`
- 분석한 하수관로 영상을 저장하는 폴더.

## 모델 파일
### `YOLOv10x.pt`
- 대표 결함을 학습한 YOLOv10x 모델.

### `YOLOv10x_all.pt`
- 대표 및 기타 결함을 모두 학습한 YOLOv10x 모델.

### `YOLOv10x_etc.pt`
- 기타 결함만을 학습한 YOLOv10x 모델.
